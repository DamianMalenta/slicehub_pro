# CANVASY — eksporty i kopie robocze

Ten katalog jest na:

- zapisane canvasy z Cursor,
- eksporty canvasów do plików,
- kopie tekstowe canvasów w `.md`, jeśli nie da się zapisać jako `.canvas.tsx`.

## Jak używać

Jeśli da się zapisać canvas do workspace:

- zapisuj go tutaj, np.:
  - `01_slicehub-three-directions.canvas.tsx`
  - `02_menu-studio-polish.canvas.tsx`
  - `03_magazyn-przeplywy.canvas.tsx`

Jeśli nie da się zapisać jako canvas:

- wklej jego treść do `.md`, np.:
  - `01_slicehub-three-directions.md`
  - `02_menu-studio-polish.md`

## Zasada nazewnictwa

Używaj prostego formatu:

- `01_nazwa-pliku`
- `02_nazwa-pliku`
- `03_nazwa-pliku`

To pozwoli czytać canvasy po kolei i łatwo je analizować.

## Ważne

To nie jest źródło prawdy dla systemu.
To jest katalog pomocniczy do:

- analiz,
- koncepcji,
- decyzji projektowych,
- handoffów do kolejnych sesji.

Jeśli z canvasa wynika ważna decyzja architektoniczna, trzeba ją potem przepisać do aktywnej dokumentacji w `_docs/`.
