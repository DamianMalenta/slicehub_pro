# ARCHIWUM — dokumenty historyczne i pomocnicze

Ten katalog zawiera pliki, które:

- były ważne w poprzednich sesjach,
- dokumentują starszy kierunek albo etap projektu,
- mogą pomóc zrozumieć historię decyzji,
- ale **nie są już głównym punktem startowym**.

## Jak używać archiwum

- Najpierw czytaj `../START_TUTAJ.md`
- Potem `../00_PAMIEC_SYSTEMU.md`
- Do archiwum zaglądaj tylko wtedy, gdy:
  - szukasz historii decyzji,
  - potrzebujesz starszego kontekstu,
  - porównujesz dawny kierunek z aktualnym.

## Co tu trafia

- stare statusy sesji,
- starsze wizje, które zostały zawężone lub zastąpione,
- pomocnicze ściągi, które nie powinny zaśmiecać głównego `_docs/`.

## Zasada

Plik w `ARCHIWUM` **nie wygrywa** z aktywnym dokumentem w głównym katalogu `_docs/`.
